The Teams feature lets learners form small groups (or be assigned to them) with private discussion topics and a shared identity. For most self-paced Liverpool Dental CPD modules this is overkill — but worth knowing about for cohort programmes.

:::caution[Not enabled here]
The Teams feature is **not enabled** on the Liverpool Dental Learning Hub. This page is reference for the day it is — raise a request via [dental.cpd@liverpool.ac.uk](mailto:dental.cpd@liverpool.ac.uk) if you need it for a cohort programme.
:::

## When teams help

- Multi-week cohort programmes (e.g. a postgraduate certificate run).
- Case-based learning where learners discuss a case in a small group.
- Peer-review workflows where you want the same group reviewing each other.

## When they hurt

- Self-paced courses where learners arrive at the team-formation page at very different times — teams stay empty.
- Single-author CPD modules with no facilitation.

## Setting teams up

1. Studio → *Settings → Advanced Settings*.
2. Set `Teams Configuration` to enable, defining team sizes and topics.
3. The Teams tab appears in the LMS course navigation.
4. Learners self-organise into teams (or you assign).

## Per-topic discussion

Each team gets a private discussion topic visible only to team members. This is the main day-to-day value of the feature.

## Pre-flight check

If you're not committing to facilitating teams for the duration of the course, leave the feature off. An empty Teams page is a worse user experience than no Teams page.
