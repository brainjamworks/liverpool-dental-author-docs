This walkthrough takes you from an empty Studio shell to a publishable CPD course in roughly 30 minutes. If you're new to Open edX, skim [What is Studio?](../what-is-studio/) and [What is the LMS?](../what-is-the-lms/) first.

![Studio Create a new course form](../../../assets/screenshots/studio-new-course.png)

*Studio's *Create a new course* form. You'll fill this in once at the start of every new CPD course.*

## You'll create

1. A course shell with title, number, run, and start date.
2. A section → subsection → unit hierarchy.
3. One text component and one multiple-choice question.
4. A publish + course-about page review.

## Steps

1. **Sign in to Studio** at `studio.learning.endo360.uk` with your author account.
2. **Create a new course.** From the dashboard, click *New Course*. Use your faculty's course code as the *Course Number* and the calendar year as the *Run* (e.g. `ENDO101` / `2026`).
3. **Add structure.** Open *Outline* and add a *Section* ("Module 1"), then a *Subsection* ("Diagnosis"), then a *Unit* ("Case overview").
4. **Add content.** Inside the unit click *Add Component → Text*, paste your introduction, and save. Then *Add Component → Problem → Multiple Choice* for a knowledge check.
5. **Publish the unit.** Click *Publish* in the unit sidebar. Set visibility to *Public* when ready.
6. **Set CPD metadata.** In *Settings → Schedule & Details* add the CPD hours, GDC development outcomes, and a course image.
